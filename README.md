# Data Structure Visualizer

## Overview

Data Structure Visualizer is an interactive web application built using HTML, CSS, and JavaScript. It helps users understand the working of various data structures through visual representation and real-time operations.

## Features

### Linear Data Structures

* Array

  * Insert
  * Delete
  * Search
  * Display

* Stack

  * Push
  * Pop
  * Peek
  * Display

* Queue

  * Enqueue
  * Dequeue
  * Front Element
  * Display

* Linked List

  * Insert Node
  * Delete Node
  * Search Node
  * Display List

### Non-Linear Data Structures

* Tree

  * Insert Node
  * Delete Node
  * Search Node
  * Traversal Visualization

* Graph

  * Add Vertex
  * Add Edge
  * Remove Edge
  * Traversal Visualization

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `web.html` in a browser or use Live Server.
4. Start exploring different data structures and their operations.

## Learning Outcomes

* Understand the behavior of linear and non-linear data structures.
* Visualize insertion, deletion, searching, and traversal operations.
* Improve understanding of Data Structures concepts through interactive learning.
